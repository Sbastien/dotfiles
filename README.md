# Dotfiles

## Installing

- Setup Macos : <https://github.com/Sbastien/macos-setup>
- Backup all your dotfiles
- Create your gitconfig.local

    ```shell
    vi ~/.gitconfig.local
    ```

    Then add your informations

    ```shell
    [user]
      name = "Your Name"
      email = "your@email"
    ```

- Run `./install`
